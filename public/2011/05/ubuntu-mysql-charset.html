Ubuntu中配置Mysql编码

Linux

找到配置文件

我是通过sudo apt-get install mysql来安装的。mysql的配置文件在/etc/mysql/my.cnf
如果找不到这个文件，可以运行sudo find / -iname ‘*.cnf’查找所有的cnf文件

修改配置文件

在[mysqld]下添加
default-character-set=utf8
在[client]下添加
default-character-set=utf8

重启mysql

sudo service mysql restart

登录mysql查看是否成功

# mysql -u root

mysql> show variables like 'char%';
