目前这个扩展是我用photoshop叠加了几个图片做出来的，自我感觉没那么难看啊。。 主要表达下载图片的意思。。你觉得呢？ 另外，喜欢这个扩展的人有设计师么，或者身边有设计师朋友么，能不能帮这个扩展设计一个图标？
作为回报，如果扩展使用了你的图标，会在选项中加入你的链接标注图标是由你提供的，虽然目前这个扩展的用户不多，但我觉得还算是一个好用的扩展吧。。 图标可以通过邮件发送给我，我的邮箱：cyk.cn#qq.com(把#换成@)
