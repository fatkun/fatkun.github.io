如何找到QQ头像地址？

  1. 首先想到的是通过搜索引擎搜索是否有现成的。不过第一次使用的关键词是“QQ头像地址”，结果找到的都失效了（如Myqq以及Vipqq的）
  2. 查看myqq以及Vipqq的代码。失望啊，它头像获取的地址是http://my.qq.com/qq_face.php，可惜，没有参数给我们调用，估计是通过Cookies传值的或者其他
  3. 想想还有没有同类的网站显示QQ头像的。对，WebQQ，里面有QQ头像地址，果然，就给我找到了。
  4. 找到之后，觉得我第一次的搜索关键字好像不太好，换一个“获取QQ头像”，丫的，搜索出来了。
