Mysql指定编码创建数据库

2010-06-21
数据库
约 84 字 预计阅读 1 分钟

文章目录

mysql 创建数据库时指定编码很重要，一般都指定为统一的UTF-8编码，如果不指定编码，默认编码为lan1，插入中文时是会提示错误的，类似”\x22\x34″

CREATE DATABASE `test2` DEFAULT CHARACTER SET utf8 COLLATE utf8_general_ci
