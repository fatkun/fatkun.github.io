男女同学交往的好处

2010-01-04
心理学
约 1004 字 预计阅读 3 分钟

我们都有过这种体验：有异性参加的活动，较之只有同性参加的活动，我们一般会感到更愉快，活动的积极性会更高，往往玩得更起劲，干得更出色。这就是心理学上的“异性效应“。当有异性参加活动时，异性间心理接近的需要就得到了满足，于是，彼此间就获得了不同程度的愉悦感，激发起内在的积极性和创造力。尽管健康的两性交往对我们的成长有诸多的好处，我们要把握好两性交往的尺度，防止“过“与“不及“。
